# <a name="remove-a-legacy-authorized-host-in-lync-server-2013-optional"></a>Lync Server 2013 での従来の承認済みホストの削除 (オプション)

<div data-xmlns="http://www.w3.org/1999/xhtml">

<div class="topic" data-xmlns="http://www.w3.org/1999/xhtml" data-msxsl="urn:schemas-microsoft-com:xslt" data-cs="https://msdn.microsoft.com/">

<div data-asp="https://msdn2.microsoft.com/asp">



</div>

<div id="mainSection">

<div id="mainBody">

<span> </span>

_**最終更新日:** 2012-09-05_

Lync Server の展開にリモートの通話コントロールを移行する場合は、従来の展開の SIP/CSTA ゲートウェイの、従来の承認済みホストエントリ (Lync Server の *信頼済みアプリケーションエントリ* ) を削除する必要があります。 管理ツールを使用して、これらの展開から承認されたホストエントリを削除する必要があります。
